# Installation

The `FAIR CLI` requires [python 3.8+](https://www.python.org/downloads/) and can be installed using [pip:](https://pypi.org/project/fair-cli/)

```
pip3 install fair-cli
```

***Note:***
On windows the command may be:
```
pip install fair-cli
```
